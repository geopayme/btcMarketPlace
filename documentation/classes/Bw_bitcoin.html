\Bw_bitcoin

Bitcoin Library

This library is a socket for the JSON RPC interface. Configuration is loaded from ./application/config/bitcoin.php The class contains functions for bitcoind and functions for bitcoind to callback in order to track information about new transactions. Also contains a function to update exchange rates from the selected provider.

signrawtransaction()

signrawtransaction(string $transaction_hex, array $inputs, array $privkeys) : array

Sign Raw Transaction

This function when provided with only $transaction_hex will attempt to sign the transaction using any keys found in the wallet. If the transaction is unsigned, then the inputs are required. $inputs is an array. If the wallet does not have the private key, then it will need to be supplied as a third parameter. A private key, or several, are given as an array. Returns an array containing: hex: a string - the transaction hex complete: boolean - indicating whether the transaction is fully signed.

Parameters

string $transaction_hex
array $inputs
array $privkeys

Returns

array

addmultisigaddress()

addmultisigaddress(int $m, array $public_keys, \(opt)array $account) : string

Add Multisig Address

This function is used to import a multisignature address into the bitcoin wallet. This is required by users when signing an unsigned/partially-signed transaction. $m determines how many keys out of the total are needed to redeem funds. $public_keys is an array containing the public keys. Order is important. $account - defaults to the main account.
