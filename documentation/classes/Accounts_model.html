get()

get(string $identifier, \array(optional) $opt) : \array/FALSE

Load an account

Will determine which columns to select from the database. If $opt['own'] == TRUE, we get more details than a normal account. Then $identifier is used to select the information information using either the hash/name/id. If the record is found, format and load more information. If the user has PGP, then add that to the returned array if successful. If there's no record, return false.

\array(optional) $opt
