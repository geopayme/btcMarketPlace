check_if_expected_spend()

check_if_expected_spend(array $output) : boolean

Check If Expected Spend

This function takes a transactions ['vout'] array and checks if this matches what we expected. Only done for order payments: as the payments leaving these addresses would have been created by the site, and have explicitly agreed outputs, any deviation in the hash means the transaction has been tampered with.

Also done to verify partially signed transactions.
