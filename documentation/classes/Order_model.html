order_finalized_callback()

order_finalized_callback(array $array)

Order Finalized Callback

This function is called with an array of information when an input in an order has been spent. If this happens, it either corresponds to an escrow or up-front payment going through. We need to check that the spend was expected - matches a hash of the expected outcome for that input we store on transaction creation. It can also happen when the order is disputed - in which case it simply progresses to complete.

Updates order information where necessary.

Parameters

array $array

Tags
