$encrypt_private_messages

$encrypt_private_messages

Encrypt Private Messages

Private messages can be encrypted with RSA keys to store them securely until the user log's in. Private key's are protected by a password, generated from the users salt and a chosen PIN. The password is never stored on the server.

$global_proxy_url

$global_proxy_url

Global Proxy URL

This setting is for the host:port which is to be used when setting a proxy. This will be used by the version checking job, and also the exchange rates job. The default is unset, as the $global_proxy_type parameter is disabled.

$settings_preserve

$settings_preserve

Settings Preserve

This entry is a place to store a backup of the sites configuration as it enters maintenance mode. This could be up to the admin, or can be triggered by a bitcoind alert or by the source code checker. Default is this is empty.

$location_list_source

$location_list_source

Location List Source

This is a string which determines what list of locations will be used throughout the program. Options are 'Default' for the custom list of locations (list of countries), but 'Custom' allows a predefined, heirarchal tree of locations to be built.

Default value is 'Default'.
